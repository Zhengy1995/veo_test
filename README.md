# Recommand Node.js Version
```bash
v18-v21
```
# Install PNPM golbally
```bash
npm i -g pnpm
```

# Verify PNPM
```bash
pnpm -v
```

# Install Dependencies
```bash
pnpm i
```

# Start Server
```bash
pnpm dev
```

# Visit `https://localhost:5173` in Browser
## Caution: The protocol must be `https`! Otherwise you can't get your location in the application!
